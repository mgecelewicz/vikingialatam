# VikingIALatam - Portfolio Inmersivo Premium

Este repositorio contiene la estructura completa para la experiencia web inmersiva de **VikingIALatam**.

## Características del Motor Gráfico
- **Carga y Descompresión en Nube**: El video se transmite automáticamente a través de la red de entrega de contenido (CDN) de GitHub de forma segura (CORS validado).
- **Control de Scroll de Alta Definición**: Mapeo preciso frame a frame vinculando los gestos táctiles de dispositivos móviles (como iPhone 16) y scroll de escritorio.
- **Transición Post-Hero Ininterrumpida**: El lienzo retiene de forma fija el último cuadro renderizado de la transformación digital para servir de fondo interactivo a la marca y llamados a la acción sin cortes negros ni parpadeos.

## Instrucciones de Despliegue en GitHub Pages
1. Asegúrate de tener el archivo `index.html` en la rama principal (`main`).
2. Ve a la pestaña de **Settings** (Configuración) de este repositorio.
3. En el menú lateral izquierdo, selecciona **Pages**.
4. En la sección *Build and deployment*, configura la fuente (*Source*) a **Deploy from a branch**.
5. Selecciona la rama `main` y la carpeta raíz (`/root`), luego haz clic en **Save**.
6. ¡Listo! En un par de minutos tu sitio web estará publicado de manera global y responsiva.
